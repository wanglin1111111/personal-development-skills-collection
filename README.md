## 💡 核心认知汇总

### 极限目标实现
**核心价值**: 极限目标不是冒险，而是科学管理不确定性。系统性准备 + 阶段化推进 + 风险前置管理 = 极限目标实现。

**精神内核**: "活出人样"（父亲遗训）、"向死而生"（生命哲学）、"贫穷是病，思想、精神、物质皆不可贫"

**方法论本质**: 从个人挑战经验中提炼系统性方法论，适用于创业、科研、体育、艺术等极限挑战场景。

### 供应链博弈
**核心价值**: 议价权不是天生的，而是通过产能锁定、客户多元化、成本壁垒、技术卡位、需求刚性五力综合构建。卡位"最窄河道"（产业链关键瓶颈节点）是议价权最强的位置。

**认知突破**:
- 传统: 大客户=稳定订单=安全 | 新认知: 大客户依赖=议价权弱势=风险
- 传统: 规模越大越安全 | 新认知: 产能锁定率决定安全度
- 传统: 成本是竞争结果 | 新认知: 成本壁垒是议价权基础

### 职场高情商评价
**黄金法则**: 捧虚不捧实、捧抽象不捧具体

2. 品性：您为人正直，做事公道
3. 智慧：您的见解很深刻
4. 威权：您在这个领域是权威
5. 健康：祝您身体健康
6. 才情：您的作品很有创意
7. 领导能力：您的领导能力很强
8. 涵养气度：您涵养深厚
9. 审美能力：您的眼光真好
10. 颜值：您很帅气/靓丽

**核心价值**: 通过一问一答的深度访谈，挖掘用户真实需求而非表面需求，避免"用户说要马车，实际需要更快到达目的地"的误解。
